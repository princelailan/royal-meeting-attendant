Royal Installation

  • System Requirements: Windows 10/11 (64-bit), a browser for Zoom/Teams/Meet, and admin privileges
  • Installer: Install Now

First Court Session

Run the EXE. A princess_config.json file appears. Edit royal_schedule.csv like so:

Time,Meeting Name,Meeting Link,Notification (minutes)
"14:00","Council of Finance","https://meet.google.com/xyz",15
"16:30","Royal Ball Planning","https://zoom.us/j/123",10
